# Lap-trinh-ung-dung-Chat-Console-via-TCP

I.Giới thiệu:

1.Hệ thống:

Ứng dụng Chat Console qua TCP là chương trình cho phép nhiều client kết nối đến một sever để trao đổi tin nhắn thông qua giao diện dòng lệnh.
Các chức năng hệ thống:

...

Hệ thống sử dụng mô hình Client - Sever

Sever:

-Lắng nghe kết nối từ client.

-Quản lí danh sách client.

-Xử lý command.

-Chuyển tiếp tin nhắn.

Client:

-Kết nối đến sever.

-Gửi dòng lệnh.

-Nhận và hiển thị tin nhắn.

2.Công nghệ sử dụng:

-Ngôn ngữ lập trình: python.

-Giao thức mạng: TCP

-Môi trường chạy: VS code
